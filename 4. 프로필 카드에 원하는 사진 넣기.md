# 4. 프로필 카드에 원하는 사진 넣기

> : 이미지를 보여주기 위해 **`<img />`** 태그를 사용
>
> :  **`src`** 라는 코드를 함께 사용해서 보여주고 싶은 사진 파일을  추가하기

```html
<img src="https://wecode-media.s3.ap-northeast-2.amazonaws.com
/users-code-image-media/487a5411-2c02-4cd5-819e-e1aefeb65a60"
/>
```
