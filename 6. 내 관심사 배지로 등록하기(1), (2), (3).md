# 6. 내 관심사 배지로 등록하기(1), (2), (3)

: 관심사 하나 하나를 목록으로 나타낼 수 있다.

: 목록은 `<li>` 태그로 만들 수 있다.

> <li> 태그란? : 목록(리스트)를 만들어주는 태그로, 사용하면 앞에 점이 찍혀있는 목록을 만들 수 있다.

```html
<li>FE DEVELOPER</li>
<li>MOBILITY</li>
<li>AI</li>
```
